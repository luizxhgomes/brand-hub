REGRAS DE USO

O que fazer e o que nunca fazer com a identidade visual

FAZER

  • Usar dark mode como padrão (Midnight #0A0E1A)
  • Manter contraste mínimo 4.5:1 em todo texto
  • Usar Montserrat 900 (Black) em MAIÚSCULAS para títulos display
  • Aplicar gradiente accent em títulos hero e CTAs
  • Usar Montserrat 500 com letter-spacing para dados e métricas
  • Manter padding mínimo de 48px nas laterais dos stories
  • Usar fotos reais (selfies, tela, setup) nos bastidores
  • Manter identidade visual consistente entre séries
  • Usar iconografia Lucide (stroke 1.5px, cor cyan)
  • Respeitar safe zones dos stories (top 80px, bottom 200px)

NÃO FAZER

  • Usar fundo branco como padrão (light mode só em LinkedIn mock)
  • Misturar mais de 3 cores de marca no mesmo slide
  • Usar peso 900 para texto corrido (reservar para títulos display)
  • Colocar emojis como ícones principais
  • Usar fotos de banco de imagem genéricas
  • Mudar paleta de cores entre posts da mesma série
  • Usar sombras coloridas excessivas (glow sutil apenas)
  • Textos com mais de 40 palavras por slide de carrossel
  • Animações bouncing, zoom excessivo ou rotação
  • Fontes decorativas ou script em qualquer contexto
